# Preparación Certificación Excel MO-210

## Angelo Zurita

Este repositorio contiene ejercicios prácticos para la preparación de la **Certificación Microsoft Excel - MO-210**.

## 📋 Contenido

El repositorio incluye 5 archivos de Excel con ejercicios diseñados para practicar las competencias evaluadas en el examen MO-210

- `MO-210_1.xlsx` - Ejercicio 1
- `MO-210_2.xlsx` - Ejercicio 2
- `MO-210_3.xlsx` - Ejercicio 3
- `MO-210_4.xlsx` - Ejercicio 4
- `MO-210_5.xlsx` - Ejercicio 5

## 🎯 Objetivos

Los ejercicios cubren las áreas principales del examen MO-210:

- Gestión de opciones y configuraciones de libros
- Gestión y formato de datos
- Creación de fórmulas y funciones avanzadas
- Creación de gráficos y objetos
- Análisis y visualización de datos
